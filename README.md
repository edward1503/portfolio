# Edward Portfolio

Next.js portfolio for Nguyen Don Duc, also known as Edward.

Live site: https://edward1503.github.io/portfolio/

## Local development

```bash
npm install
npm run dev
```

## Static GitHub Pages build

```bash
$env:GITHUB_PAGES='true'; npm run build
```

The GitHub Actions workflow builds the static export and publishes `out/` to the `gh-pages` branch.
